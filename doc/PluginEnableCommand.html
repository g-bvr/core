PluginEnableCommand

Purpose

This command enables a plugin. The class files must be present in the class path (e.g. as result of a PLUGIN COMPILE command executed before this).

Variants

UsageDescription
PLUGIN ENABLE plugin This command enables a plugin. The class files must be present in the class path (e.g. as result of a PLUGIN COMPILE command executed before this).

Arguments

ArgumentValue
plugin The name of the plugin to be enabled.

Plugin

This command is defined in CorePlugin
