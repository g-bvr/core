PluginInstallCommand

Purpose

This command executes system level installation steps required by a plugin. If there are any installation steps to be executed on system level is defined by the method Plugin.getCommands(). These steps are supposed to be executed in a shell script while setting up a docker container.

Variants

UsageDescription
PLUGIN INSTALL plugin This command executes system level installation steps required by a plugin. If there are any installation steps to be executed on system level is defined by the method Plugin.getCommands(). These steps are supposed to be executed in a shell script while setting up a docker container.

Arguments

ArgumentValue
plugin The name of the plugin for which system commands are to be executed.
